◆ wywolanieMenu()

void Menu::wywolanieMenu ( )

Odpowiada za obsługę interfejsu.

Wywołuje funkcję menu() i w zależności od wyboru użytkownika, tworzy obiekty i wywołuję funkcje z poszczególnych klas.

Attention
jeśli użytkownik wybierze opcję pierwszą to wykona się operacja kodowania za pomocą funkcji Koder::kodowanie() z klasy Koder.
jeśli użytkownik wybierze opcję drugą to wykona się operacja dekodowania za pomocą funkcji Dekoder::dekodowanie() z klasy Dekoder.
jeśli użytkownik wybierze opcję trzecią to program wyświetli zawartość plików gdzie znajduje się tekst odkodowany i zakodowany przy pomocy funkcji Pliki::odczyt() z klasy Pliki.
jeśli użytkownik wybierze opcję czwartą to zakończy działania programu wyświetlając komunikat.
jeśli użytkownik wprowadzi z klawiatury jakąkolwiek inną cyfrę to wyświetli informację to błednym wyborze.

◆ zapytanieZapis()

char Menu::zapytanieZapis ( )

Pyta użytkownika czy zapisać tekst do plików.

W pętli pyta użytkownika czy chciałby zapisać tekst do plików, musi wprowadzić 't' lub 'n'.

Returns
Zwraca literę podaną przez użytkownika.
Attention
powtarza zapytanie dopóki użytkownik nie wprowadzi 't' lub 'n'.
